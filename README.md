<h3>课程介绍：</h3>
<p>本次培训从系统安全机制到应用运行机制讲解安卓安全等理论知识，介绍分析安卓APP常见安全漏洞以及如何修复防范。影响千万安卓应用的寄生兽漏洞实际案例分析，攻击常见的流行APP，如何打开一张图片黑掉本地应用，以及漏洞如何防范。安卓远程意图协议漏洞实际案例分析，介绍多种不为人知的真实远程攻击漏洞，如何通过点击一个链接黑掉本地应用，以及漏洞如何防范。</p>
